# DevOps Command Center

A full-stack enterprise-style platform designed to simulate real-world IT Operations and DevOps workflows.

The platform provides incident management, change management, deployment tracking, system monitoring, and audit capabilities while demonstrating modern DevOps engineering practices.

## Project Goals

This project was built to demonstrate skills in:

* Backend API Development
* Database Design
* Infrastructure as Code
* Containerization
* CI/CD Automation
* Kubernetes Operations
* Monitoring & Observability
* Platform Engineering

## Current Architecture

```text
FastAPI
   |
SQLAlchemy
   |
PostgreSQL
```
